# ControlPlane

## 介绍
针对私有集群下模拟过程中所面临的模型管理相关需求，设计面向私有集群的模型控制平面（提供自动化部署、迁移和容错处理等功能的私有集群管理平台）基础架构，为组织机构已有的集群架构模式提供轻量化管理方案，并以模型服务化调用的方式供模型使用者使用。
